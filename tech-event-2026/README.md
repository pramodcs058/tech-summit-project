# Tech Event 2026

Welcome to the Tech Event 2026 website project! This project is designed to provide information and facilitate registration for the upcoming Tech Event 2026.

## Project Structure

The project consists of the following files and directories:

- **index.html**: The main landing page of the Tech Event 2026 website, featuring the navigation bar and links to other pages.
- **registration.html**: A responsive registration page with a login form that includes email validation and a password strength check.
- **events.html**: A page highlighting the key events of Tech Event 2026, providing details about each event.
- **location.html**: A page with information about the event location, including maps and directions.
- **sales.html**: A page outlining ticket sales information, pricing, and purchasing options for the event.
- **contact.html**: A page that includes a contact form and details for attendees to reach out with inquiries.

### CSS Files

- **css/styles.css**: Contains the main styles for the website, defining layout, typography, and design elements.
- **css/responsive.css**: Includes media queries and styles to ensure the website is responsive and looks good on various devices.

### JavaScript Files

- **js/validation.js**: Contains JavaScript functions for validating the registration form inputs, including email format and password strength checks.
- **js/main.js**: Includes general JavaScript functions for the website, such as handling navigation and other interactive elements.

### Assets

- **assets/fonts**: A directory containing font files used throughout the website.

## Setup Instructions

1. Clone the repository to your local machine.
2. Open the project folder in your preferred code editor.
3. Open `index.html` in a web browser to view the main landing page.
4. Navigate through the website using the provided links in the navigation bar.

## Features

- Responsive design for optimal viewing on various devices.
- User-friendly registration form with validation.
- Detailed information about key events, location, ticket sales, and contact options.
